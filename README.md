# Pesquisa de Validação - Autfisca

Página pública da pesquisa de validação de mercado da Autfisca.

A Autfisca é uma solução em desenvolvimento voltada à inteligência fiscal assistida, organização de documentos fiscais, saneamento cadastral, pré-validação de inconsistências e redução de retrabalho em rotinas fiscais.

As respostas são enviadas ao Google Sheets via Google Apps Script.
